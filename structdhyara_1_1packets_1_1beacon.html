Detailed Description

Each node \(X_{i}\) broadcasts a Beacon at a fixed interval.

The Beacon contains Time at \(X_{i}\) ( \(T_{i}\)) at the time of sending. The Beacon is received by all \(X_{j}\) in the neighbourhood. The neighbours \(\forall X_{j} \in Neibourhood(X_{i})\) respond with an Acknowledgement, which contains the same \(T_{i}\) which was received in the Beacon.

+---- 8 bytes -----+---- 1 byte -----+---- N bytes ---+
| Time at Source | ssid length (N) | Name |
