parses the argument at position i in the arguments list.

Parses the argument at *i. If that matches with the param in this node of the chain then looks for the value in the same argument (e.g. -t20) or advances the iterator to the next argument and tries to use *(i+1) as the value of the param (e.g. -t 20). returns the next of last matched arg's iterator which are i+1 and i+2 in these two cases. If the param in this node of chain is not matched then returns i.
