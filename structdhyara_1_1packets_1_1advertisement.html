Detailed Description

If any operation on the routing table \(R_{i}\) alters/creates a best route or the delay of a best route for a destination \(X_{t}\) then an Advertisement is sent to \(X_{j}\,\forall X_{j} \in Neibourhood(X_{i})\) from \(X_{i}\).

The Advertisement contains the destination node \(X_{t}\) and \(\delta_{t,i}\) the minimum delay to reach \(X_{t}\) from \(X_{i}\). The best intermediate node (which may be 0 also) is not advertised. If the best vector is not altered by the updatation of routing table then no Advertisement packet is sent. However if \(\exists X_{k} ∊ X\) for which no broadcast packet has been sent for a large amount of time then an Advertisement regarding \(X_{k}\) is sent even if there is not change in the best vector. The advertisement packet contains route information about a node (dest). The route information includes one trip delay from the source node.

+--- 6 bytes ---+---- 8 bytes -----+----- 1 byte -----+---- 1 byte -----+---- N bytes ---+
| destination | one trip delay | hops in one trip | name length (N) | Name |
+---------------+------------------+------------------+-----------------+----------------+
