Detailed Description

The dhyara ad-hoc network.

The network is started using the start method. All related networking tasks and actions are created and maintained by the network.

void dhyara::network::isolate ( const dhyara::address &  x,
const dhyara::address &  y 
)

Isolate two nodes from each other.

The isolated nodes won't be able to communicate with each other directly. However they will be able to communicate via intermediate nodes. This function can be used to force multi-hop communication between two nodes even when they can communicate directly.

Attention
isolation is achieved by banning x from acknowledgeing y's beacon and banning y from acknowledging y's beacon.
It is desirable but not necessary to perform isolation before the network is started.
Parameters
xthe first node
ythe second node
